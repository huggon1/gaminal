# Snake

A terminal snake game.

## Requirements

- `uv`
- Python 3.12+
- A terminal with Textual support

## Setup

```bash
uv sync
```

## Run

```bash
uv run python -m snake --theme modern
uv run python -m snake --theme stealth
```

Press `t` while playing to switch theme.

## Controls

- Arrow keys / `WASD`: move snake
- `Space`: pause/resume
- `r`: restart
- `q`: quit
