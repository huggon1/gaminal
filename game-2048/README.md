# 2048

A terminal implementation of 2048.

## Requirements

- `uv`
- Python 3.12+
- A terminal with Textual support

## Setup

```bash
uv sync
```

## Run

```bash
uv run python -m game_2048 --theme modern
uv run python -m game_2048 --theme stealth
```

Press `t` while playing to switch theme.

## Controls

- Arrow keys / `WASD`: move tiles
- `r`: restart
- `q`: quit
